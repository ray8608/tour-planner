# ✈️ 旅遊行程規劃工具

一個純瀏覽器的旅遊行程規劃工具，無需安裝、無需帳號，開啟即用。支援雲端共用，可與家人或朋友共同查看行程。

## 功能總覽

### 🗓 行程規劃
- **多行程管理**：點頂部「📋 行程」按鈕可新增、切換、刪除多個旅行計畫，各自獨立儲存
- **旅行日期**：設定出發日期後，每天標籤自動顯示實際日期（如 `6/15(日)`），方便確認星期幾
- **多天行程**：可新增多個天數，每天獨立管理景點
- **景點管理**：新增、刪除景點，可設定名稱、停留時間、備注
- **交通方式**：景點之間可選擇步行、開車、大眾運輸等交通方式，並設定預估交通時間
- **時間軸**：自動根據出發時間 + 停留時間 + 交通時間計算每個景點的抵達與離開時間
- **拖曳排序**：直接拖曳景點卡片調整順序
- **飯店設定**：每天可設定出發地（飯店）與目的地（飯店）名稱

### 📁 匯入 / 匯出
- **匯出 JSON**：將目前行程儲存為 `.json` 檔案備份
- **匯入 JSON**：從備份的 `.json` 檔案還原行程

### ☁️ 雲端共用
行程可上傳至雲端，分享給家人或朋友查看、匯入。共用區分為**公開**與**非公開**兩種模式。

#### 公開行程
所有人都可以在公開分頁看到，適合完全公開分享。

#### 非公開行程（暗號分組）
只有知道暗號的人才能查詢，適合家族旅遊或小群體共用。

**範例：**
> 家族北海道旅遊，全家人共用一個暗號 `hokkaido2025`。
> 爸爸上傳行程時選「非公開」並輸入暗號 `hokkaido2025`；
> 其他家人在非公開分頁輸入同樣的暗號 `hokkaido2025`，就能看到爸爸分享的行程。
> 同一個暗號可以有多筆行程，方便不同人各自上傳版本再互相比較。

上傳時需設定**刪除密碼**，之後若要修改（覆蓋）或刪除該筆雲端行程，需輸入當初設定的密碼才能操作。

點行程旁的**「管理」**按鈕可進行：
- **更換密碼**：輸入舊密碼驗證後設定新密碼
- **刪除行程**：輸入密碼確認後從雲端移除

上傳成功後會自動產生**分享連結**（`?import=...`），可直接傳給對方；對方開啟連結即會跳出確認視窗並自動匯入行程。點行程旁的**「分享」**按鈕可隨時重新複製連結，不需重新上傳。

### ⚙️ 設定
- **版面配置**：A 頁籤 / B 側邊欄 / C 垂直捲動，三種版面隨喜好切換
- **主題**：淺色 / 深色 / 跟隨系統
- **預設交通方式**：設定新景點之間預設的交通方式

## 技術說明

| 項目 | 說明 |
|---|---|
| 前端 | 純 HTML + CSS + JavaScript（無框架） |
| 資料儲存 | `localStorage`（本地），Firebase Firestore（雲端共用） |
| 暗號安全性 | 以 SHA-256 hash 儲存，不儲存明文 |
