## Phase 3：JS 改動 — 觸控拖曳修正

### Task 3.1 — 修改 `touchstart` 改為長按 200ms 啟動

找到第 2426–2450 行（`// --- Touch Drag (mobile) ---` 區塊），**完整替換** `touchDrag`/`touchGhost` 宣告及 `touchstart` listener：

現有程式碼（第 2426–2450 行）：
```javascript
// --- Touch Drag (mobile) ---
let touchDrag  = null;
let touchGhost = null;

app.addEventListener('touchstart', e => {
  const handle = e.target.closest('.drag-handle');
  if (!handle) return;
  const card = handle.closest('.spot-card');
  if (!card) return;
  e.preventDefault();
  const touch = e.touches[0];
  const rect  = card.getBoundingClientRect();
  touchDrag = {
    spotId:    card.dataset.spotId,
    fromDayId: card.dataset.dayId,
    offsetX:   touch.clientX - rect.left,
    offsetY:   touch.clientY - rect.top
  };
  touchGhost = card.cloneNode(true);
  touchGhost.style.cssText = `position:fixed;z-index:9999;pointer-events:none;opacity:0.8;` +
    `width:${rect.width}px;left:${touch.clientX - touchDrag.offsetX}px;` +
    `top:${touch.clientY - touchDrag.offsetY}px;box-shadow:0 8px 24px rgba(0,0,0,0.4);`;
  document.body.appendChild(touchGhost);
  card.classList.add('dragging');
}, { passive: false });
```

替換為：
```javascript
// --- Touch Drag (mobile) ---
let touchDrag       = null;
let touchGhost      = null;
let touchDragTimer  = null;   // 長按計時器
let touchStartData  = null;   // touchstart 暫存資料（計時器啟動前）

app.addEventListener('touchstart', e => {
  const handle = e.target.closest('.drag-handle');
  if (!handle) return;
  const card = handle.closest('.spot-card');
  if (!card) return;
  // 不在 touchstart 立即 preventDefault，讓瀏覽器可以先處理捲動
  const touch = e.touches[0];
  const rect  = card.getBoundingClientRect();
  touchStartData = {
    spotId:    card.dataset.spotId,
    fromDayId: card.dataset.dayId,
    offsetX:   touch.clientX - rect.left,
    offsetY:   touch.clientY - rect.top,
    startX:    touch.clientX,
    startY:    touch.clientY,
    card,
    rect
  };
  // 長按 200ms 後才啟動拖曳
  touchDragTimer = setTimeout(() => {
    if (!touchStartData) return;
    const { card: c, rect: r, offsetX, offsetY, spotId, fromDayId } = touchStartData;
    const t = touchStartData;
    touchDrag = { spotId, fromDayId, offsetX, offsetY };
    touchGhost = c.cloneNode(true);
    // iOS scroll offset 修正：使用 pageY（= clientY + scrollY）減去 offsetY，
    // ghost 設為 position:fixed 時 top 用 clientY；但 ghost 在 body 上 position:absolute
    // 時需要 pageY。此處使用 position:fixed + clientY，因為 ghost 覆蓋 viewport。
    touchGhost.style.cssText = `position:fixed;z-index:9999;pointer-events:none;opacity:0.8;` +
      `width:${r.width}px;left:${t.startX - offsetX}px;` +
      `top:${t.startY - offsetY}px;box-shadow:0 8px 24px rgba(0,0,0,0.4);`;
    document.body.appendChild(touchGhost);
    c.classList.add('dragging');
    touchStartData = null;
  }, 200);
}, { passive: true });
```

**說明：**
- `touchstart` listener 改為 `{ passive: true }`，不阻止預設捲動行為。
- 200ms 計時器到期後才建立 `touchDrag` 和 ghost，進入拖曳模式。
- `touchStartData` 暫存 touchstart 時的資料，供計時器回呼使用。

### Task 3.2 — 修改 `touchmove`：計時器取消邏輯 + iOS ghost 位置修正

找到第 2452–2469 行，現有程式碼：
```javascript
app.addEventListener('touchmove', e => {
  if (!touchDrag || !touchGhost) return;
  e.preventDefault();
  const touch = e.touches[0];
  touchGhost.style.left = (touch.clientX - touchDrag.offsetX) + 'px';
  touchGhost.style.top  = (touch.clientY - touchDrag.offsetY) + 'px';

  touchGhost.style.display = 'none';
  const el = document.elementFromPoint(touch.clientX, touch.clientY);
  touchGhost.style.display = '';

  document.querySelectorAll('.drag-over, .drag-over-zone')
    .forEach(n => n.classList.remove('drag-over', 'drag-over-zone'));
  const overCard = el?.closest('.spot-card');
  const overZone = el?.closest('[data-drop-zone]');
  if (overCard && overCard.dataset.spotId !== touchDrag.spotId) overCard.classList.add('drag-over');
  else if (overZone && !overCard) overZone.classList.add('drag-over-zone');
}, { passive: false });
```

替換為：
```javascript
app.addEventListener('touchmove', e => {
  const touch = e.touches[0];

  // 尚未進入拖曳模式（等待長按計時器）
  if (!touchDrag) {
    if (touchStartData && touchDragTimer) {
      // 移動超過 5px 閾值 → 取消計時器，讓瀏覽器正常捲動
      const dx = touch.clientX - touchStartData.startX;
      const dy = touch.clientY - touchStartData.startY;
      if (Math.abs(dx) > 5 || Math.abs(dy) > 5) {
        clearTimeout(touchDragTimer);
        touchDragTimer = null;
        touchStartData = null;
      }
    }
    return;
  }

  // 已進入拖曳模式
  if (!touchGhost) return;
  e.preventDefault();  // 阻止捲動（只在拖曳中才呼叫）

  // iOS scroll offset 修正：ghost 是 position:fixed，用 clientX/clientY 定位 viewport 座標
  // touch.clientY 是相對 viewport 的座標，不受 scrollY 影響，正確。
  // （舊版用 clientY 本來就對，但現在確保邏輯清晰）
  touchGhost.style.left = (touch.clientX - touchDrag.offsetX) + 'px';
  touchGhost.style.top  = (touch.clientY - touchDrag.offsetY) + 'px';

  touchGhost.style.display = 'none';
  const el = document.elementFromPoint(touch.clientX, touch.clientY);
  touchGhost.style.display = '';

  document.querySelectorAll('.drag-over, .drag-over-zone')
    .forEach(n => n.classList.remove('drag-over', 'drag-over-zone'));
  const overCard = el?.closest('.spot-card');
  const overZone = el?.closest('[data-drop-zone]');
  if (overCard && overCard.dataset.spotId !== touchDrag.spotId) overCard.classList.add('drag-over');
  else if (overZone && !overCard) overZone.classList.add('drag-over-zone');
}, { passive: false });
```

**說明：**
- listener 仍是 `{ passive: false }` 以便在拖曳模式下呼叫 `e.preventDefault()`。
- 但 `e.preventDefault()` 只在 `touchDrag` 存在（已進入拖曳模式）時呼叫，避免 Chrome passive 警告實際影響。
- 移動 >5px 時清除計時器，使用者可以正常捲動頁面而不誤觸拖曳。
- Ghost 位置：`position: fixed` + `touch.clientY`（viewport 座標），不需要加 `scrollY`。pageY 的修正適用於 `position: absolute` 的情況，這裡用 fixed 所以 clientY 正確。

### Task 4.1 — 新增共用函式 `addSpotAndFocus(dayId)`

在 `add-spot` action handler（第 2195 行）**之前**，新增以下函式。建議放在第 2195 行前的空白行，即 `// --- Action Dispatcher ---` 區塊或其附近。實際上放在 `attachEvents()` 函式內部，`app.addEventListener('click', ...)` 的 **外部**（函式層級）或 action handler 段落之前均可。

最安全的插入位置：找到第 2195 行 `if (action === 'add-spot') {`，在其**前一行**（`if (action === 'delete-day') {` 結尾的 `return; }` 之後）插入：

```javascript
// 共用：新增景點並 focus 最新 spot-name
function addSpotAndFocus(dayId) {
  setState(s => {
    const day = s.days.find(d => d.id === dayId);
    if (day) day.spots.push({ id: genId(), name: '', stayDuration: 0, notes: '' });
  });
  // render() 已在 setState 內呼叫，此時 DOM 已更新
  requestAnimationFrame(() => {
    const cards = document.querySelectorAll(`.spot-card[data-day-id="${dayId}"]`);
    const last = cards[cards.length - 1];
    last?.querySelector('.spot-name')?.focus();
  });
}
```

**注意：** `addSpotAndFocus` 必須是 `attachEvents()` **外部**的函式（與 `render()`、`setState()` 同層），否則無法在 keydown listener 中呼叫。確認 `attachEvents()` 的結束 `}` 位置，將此函式放在 `attachEvents()` 定義**之前**或**之後**均可，只要在全域（`<script>` 內頂層）即可。
